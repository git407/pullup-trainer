# PullUp Trainer - Android приложение для тренировок подтягиваний

Простое Android приложение для отслеживания прогресса тренировок подтягиваний по уровням.

## Функции

- 3 уровня тренировок (по 19 дней каждый)
- Отслеживание текущего дня тренировки
- Простой интерфейс со списком всех дней
- Сохранение прогресса между сеансами

## Сборка APK

### Требования

- Java JDK 8 или выше
- Android SDK (через Android Studio или командную строку)

### Сборка через командную строку

1. Убедитесь, что у вас установлен Android SDK и переменная окружения `ANDROID_HOME` настроена
2. Запустите скрипт сборки:

**Windows:**
```bash
build-apk.bat
```

**Linux/Mac:**
```bash
./gradlew assembleRelease
```

3. APK файл будет находиться в: `app/build/outputs/apk/release/app-release.apk`

### Сборка через Android Studio

1. Откройте проект в Android Studio
2. Выберите Build → Build Bundle(s) / APK(s) → Build APK(s)
3. Или Build → Generate Signed Bundle / APK → APK

## Установка

После сборки APK, скопируйте файл `app-release.apk` на Android устройство и установите его.

**Внимание:** На Android устройствах по умолчанию запрещена установка приложений из неизвестных источников. Вам нужно разрешить установку из неизвестных источников в настройках безопасности.

## Использование

1. Откройте приложение
2. Просмотрите список всех уровней и дней тренировок
3. Нажмите на день, чтобы отметить его как текущий
4. Текущий день будет выделен зеленым цветом
5. При следующем запуске приложения выбранный день будет сохранен

## Структура тренировок

- **Уровень 1**: От 1 до 10 подтягиваний
- **Уровень 2**: От 6 до 20 подтягиваний  
- **Уровень 3**: От 18 до 30 подтягиваний

Каждый уровень содержит 19 дней тренировок с различными подходами.
